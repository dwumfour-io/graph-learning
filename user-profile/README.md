# Graph User Profile Console app

### Description
 Gets a specified attributes of a signed in users profile.

1. DisplayName
2. Email
3. JobTitle
4. Department
5. Country
6. CreatedDateTime
7. Account Enabled
8. LastPasswordChangeDateTime
